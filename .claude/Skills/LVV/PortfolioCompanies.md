## Usage in Screening

### Co-Investor Detection

When evaluating an investor, check if they've invested in LVV portfolio companies:

```
Investor "Longevity Fund X" has co-invested in:
- Fauna Bio (Series A, 2023)
- Integrated Biosciences (Seed, 2024)

Strategic Value: +2 points for existing relationship
```

### Conflict Detection

When evaluating a company, check for competitive overlap:

```
Company "SenesClear" focuses on senolytics.

Potential overlap with portfolio companies:
- SENISCA (senolytics)
- Deciduous Therapeutics (senescence)

Flag for review: Competitive or complementary?
```

### Syndicate Partners

Investors who have co-invested with LVV are higher priority:
- Existing relationship = easier syndication
- Shared thesis validation
- Potential for future deals
